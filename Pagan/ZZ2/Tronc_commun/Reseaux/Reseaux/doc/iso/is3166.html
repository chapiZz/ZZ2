This list states the country names (official short names in English) in alphabetical order as given in ISO 3166-1 and the corresponding ISO 3166-1-Alpha-2 code elements.
This list is updated whenever a change to the official code list in ISO 3166-1 is effected by the ISO 3166/MA (Maintenance Agency).
This list is complete and up-to-date as of 2002-02-20. It lists 239 official short names and code elements.
