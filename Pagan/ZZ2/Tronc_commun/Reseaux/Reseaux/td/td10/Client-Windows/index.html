WINDOWS 95 et Internet

Configuration d'une connexion en dial-up IP

Document inspiré de France-Pratique
adapté pour les TP en salle 212 de l'ISIMA


Cette procédure indique, pas par pas, la marche à suivre pour réaliser une connexion à Internet par modem à partir de WINDOWS 95.

Le script de connexion réalisé au terme de cette procédure se substitue avantageusement à TRUMPET WINSOCK qui devient donc inutile.
Vous pouvez continuer à utiliser les versions 16 bits des clients TCP-IP tels que Netscape, Eudora, ou FTP, mais vous obtiendrez des résultats plus stables en instalant sur votre machine les versions 32 bits de ces applications. Vous trouverez sur le serveur de France Pratique toutes les informations nécéssaires quant à la disponibilité et aux modalités de téléchargement de ces applications.

1ére étape : Configurer "ACCES RESEAU A DISTANCE"

2ème étape : Sélection des composantes "RESEAU" nécessaires

3ème étape : Configurer les composantes "RESEAU"

4ème étape : Terminer de configurer les composantes "RESEAU"

5ème étape : Enregistrer un script de connexion

6ème étape : Première connexion
